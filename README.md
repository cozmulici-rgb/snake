# Snake (Go, Windows)

Simple console Snake game written in Go.

## Requirements
- Windows
- Go 1.22+ installed and available on PATH

## Run
From this folder:

```powershell
go fmt ./...
go run .
```

Or use the launcher:

```powershell
.\run.bat
```

## Controls
- `W` up
- `A` left
- `S` down
- `D` right
- `Q` quit
